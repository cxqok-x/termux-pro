## 🚀 Installation
 ONE LINE COMMAND -🎯
 just copy paste this link in your termux.

```
pkg update -y && pkg install git -y && git clone https://github.com/cxqok-x/termux-pro.git && cd termux-pro && chmod +x termux-pro.sh && bash termux-pro.sh
```
